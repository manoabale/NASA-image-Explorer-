# NASA-image-Explorer-

A simple image search app using the [NASA Image and Video Library API](https://images.nasa.gov/).

## Features
- Search NASA’s image library by keyword
- Display image, title, and description
- Responsive grid layout

## Setup
1. Clone the repo.
2. Open `index.html` in your browser.
3. Or deploy to GitHub Pages.

## Deployment
- Push this repo to GitHub.
- Go to **Settings → Pages → Source → main branch → /root**.
- Your app will be live at `https://your-username.github.io/nasa-image-explorer`.
